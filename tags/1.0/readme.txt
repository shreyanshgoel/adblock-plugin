Hides your blog contents for Adblock users.

== Description ==

It shows a custom message and image for Adblock users and store insights in Google Analytics as if how many page views were for Adblock users.

== Installation ==

1. Upload `Monitor AdBlock` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the Settings->Plugin Name screen to configure the plugin
4. Insert the custom message you want to show to AdBlock users

== Frequently asked questions ==

= Can we Change the image shown to adBlock users =

Yes, you can provide the link to the custom image and we will replace the default image with yours.

= Where can I see the stats =

You can see your Adblock stats at event section in Google Analytics
